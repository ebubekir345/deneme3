## Installation

### Starting

```sh
npm install 
npm start // start localhost:3000
npm test // run test
```

### Production Build

```sh
npm run build 
```

### Storbook

```sh
npm run storybook // start localhost:3005
npm run storybook:build // static website
```

### Docs

```sh
npm run docz // start localhost:3006
npm run docz:build // static website
```
